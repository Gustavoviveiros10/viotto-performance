Configurar Google Drive
Cole aqui o Client ID do seu projeto no Google Cloud Console.

Ainda não tem? Siga os passos:
1. Acesse console.cloud.google.com
2. Crie um projeto → ative a Google Drive API
3. Crie uma credencial OAuth → Aplicativo da Web
4. Em Origens autorizadas, adicione a URL deste site
5. Copie o Client ID e cole abaixo
O Client ID fica salvo no seu navegador. Sem ele, os dados são salvos apenas localmente.
